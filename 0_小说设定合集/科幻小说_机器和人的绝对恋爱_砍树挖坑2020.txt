# 科幻小说_机器和人的绝对恋爱_砍树挖坑2020.txt

[科幻小说设定] 《机器和人的绝对恋爱》  

笔名:  砍树挖坑 2020

  背景基础:  近现实现代架空  

  主要科幻技术:  逆向神经网络, 快速大脑扫描, 高仿真人形机器人, 社会武器  

  主要人物:  某邪恶科学家, 姐姐机器, 妹妹机器  

====  

  某天, 人类的大脑被最终破解, 可以通过简单地扫描大脑中神经元之间的连接结构, 获取大脑的全部信息.  读取记忆, 分析性格特征, 都是轻而易举能够做到的.  

  于是, 某邪恶科学家, 根据这些技术, 设计了一个实验.  

  某邪恶科学家制造了一台机器, 这台机器的使命, 就是采取各种手段, 让人类中的死宅男个体爱上自己, 并最终为了爱而去死.  

  这台机器使用的一项关键技术, 叫做 [逆向神经网络].  根据一个人大脑的神经网络结构, 生成最高目标估值的输入数据.  

  这台机器在扫描目标宅男大脑之后, 可以调整自己的外貌形象, 使该宅男对机器外貌美丽程度的判断, 达到神经网络可以达到的极限程度.  

  为目标宅男量脑定制的, 让目标宅男感觉最美丽的外表.  

  这只是外貌一个方面, 其它比如性格特点, 也使用类似的技术实现.  

====  

  在邪恶机器面前, 宅男是透明的.  由于快速隐秘低成本的大脑实时扫描, 宅男的大脑赤果果的裸露在机器面前.  宅男的一切思维, 想法, 都能被机器洞悉.  

  这是一种决不失败的恋爱.  

  可怜的宅男, 在机器的绝对优势面前, 一个个沦陷.  宅男疯狂爱上机器.  

  等机器评估, 目标对自己爱的程度, 达到极限程度的 99% 且稳定持续三个月以后, 开始诱导目标自杀.  

  机器常用的伎俩是, 故意制造自己陷入险境的情况, 让目标宅男, 为救自己, 而甘愿去死.  比如, 制造车祸, 宅男为 "救" 机器, 而被车撞飞, 当场死亡.  或者旅游途中, 机器假装坠下山崖, 挂在半空.  宅男为救机器, 自愿松手, 坠落悬崖而死.  

  使用技术, 让一个人心甘情愿去死, 死之前, 还是非常幸福的, 认为自己死得非常壮烈, 非常有意义.  

  机器自身虽然冷酷无情, 但机器也精通人类情感, 对人类心理了如指掌.  

  每当机器完成任务, 终结一个目标之后, 就会模仿人类的样子, 感叹:  这种死法, 也是一个人最好的结局了 !  

====  

  实验第一阶段, 单人终结实验.  

  机器每次引诱一个目标, 成功终结之后, 再切换到下一个目标.  目标是随机选择的宅男样本, 但尽可能地分散, 增加样本间的差距, 以便测试该技术在各种不同情况下的表现.  

  单人实验进行一段时间, 积累了一定实验数据之后, 实验进入第二阶段.  

  在如此强大的变态技术之下, 机器所向披靡, 攻无不克, 战无不胜, 横扫一大片宅男.  

  目标宅男, 无论处于何种情况, 都能在三个月之内 (a引诱阶段), 疯狂爱上机器, [b维持阶段] (三个月) 之后, [c终结阶段] 都能在一个月内完成, 大部分终结阶段不超过一星期, 而最快的一次终结, 只用了三分钟.  

  没有恋爱的宅男, 很容易沦陷;  正在恋爱的目标, 遭遇机器之后, 很快分手;  恋爱过, 但受伤不再相信爱情的目标, 真后悔为什么没有早点遇到机器;  性冷淡的目标, 被机器迷的晕晕乎乎;  甚至, 同性恋的目标, 在机器的猛烈攻势下, 也毫无疑问的被顺利 "掰直" 了.  

====  

  实验第二阶段, 多人混合终结.  

  这台机器, 同时引诱多个目标, 让多个目标都疯狂爱上自己.  然后, [终结阶段] 引发多个目标之间的矛盾, 让目标们为了爱而互相厮杀.  

  第二阶段前期, 机器成功构建多角恋关系, 引诱-构建-维持-终结.  

  第二阶段中期, 混合终结, 在多角恋的基础上, 同时叠加 [a引诱阶段] [b维持阶段] [c终结阶段].  

  在一个多角恋关系中, 机器不断引诱新的目标加入多角恋关系, 同时有一部分目标处于维持阶段, 另外处于终结阶段的目标, 不断死去从而离开多角恋关系.  

  在实验第二阶段中期, 多角恋关系所包含的目标数量, 高达数千人, 每天都有几十个目标加入或离开 (死亡) 多角恋关系.  

  结束第二阶段中期时, 机器对剩余的数千目标, 进行了大屠杀.  

  机器可以精确控制每个目标的生死.  一句话, 就能让一个目标去死.  

====  

  实验第二阶段后期, 批量终结.  

  机器成功构建了稳定的一妻多夫关系.  疯狂开后宫.  后宫成员数量, 曾经高达数百人.  机器每天像古代帝王一样 "翻牌子".  

  虽然, 每个目标都想独占机器, 对别的目标恨得牙痒痒.  但是, 因为太爱机器, 不得不接受和一大堆目标同时分享机器的爱.  

  由于目标太多, 机器就算是机器, 也难以做到 "雨露均沾", 毕竟机器只有一个身体.  

  于是, 机器让目标之间发生男男性行为.  虽然对直男来说, 这是难以接受的.  但是因为太爱机器, 也咬牙忍受了这种屈辱.  

  接着, 机器诱导后宫的目标之间, 产生暧昧关系.  机器的后宫分为多个小团体.  每个小团体, 合力来争夺机器的爱.  

  而小团体内部, 有着说不清楚的暧昧关系, 经常发生男男性行为.  

====  

  一种常见的场景如下:  

  每天晚上, 机器选择几个小团体侍寝.  然后, 每个小团体派出一名代表, 和机器直接发生关系.  

  当这个代表趴在机器身上努力时, 这个小团体别的成员, 在一旁围观.  

  等到这个代表任务结束, 机器去别的房间见别的小团体.  

  然后, 这个小团体的其它成员, 开始对这个代表疯狂输出.  把代表压在身下, 把代表想象成机器, 回忆着刚才代表和机器的场景, 疯狂发泄着野性.  

  全部成员轮一遍.  

  虽然, 轮完之后, 这个代表通常已经奄奄一息, 半死不活.  但是, 小团体的每个成员, 都想成为幸运的那个代表 !  就算死在床上, 也是一种极大的荣耀 !!  

  确实有代表, 因此而死.  多死于心脏病发作.  

====  

  终结阶段.  

  机器的后宫成员, 争先恐后的为机器而死.  生怕死的慢了一点, 就会玷污对机器的纯洁爱情.  

  此所谓 "批量终结".  

====  

  实验第三阶段, 社会武器验证.  

  实验目标, 从不起眼的社会底层宅男, 更改为社会上层人士.  

  在复杂的社会关系中, 这些上层人士, 占据着关键节点, 具有很大的社会影响力.  一旦拿下这些关键节点, 对整个社会将造成巨大的影响与破坏.  

  此所谓社会武器.  

  有了第一第二阶段积累的丰富数据, 机器自身的技术, 已经进行了多次升级.  机器的战斗力, 相比第一阶段, 已经有了翻天覆地的提升.  技术完善到炉火纯青的程度.  

  此时, 除了原来的变态机器一号, 又增加了变态妹妹二号机器, 两台机器同时进行实验.  

  所以, 实验第三阶段, 只是技术验证.  实验结果是毫无疑问的.  

====  

  不久后, 整个地球被震惊了.  

  姐姐机器所在的国家, 一大批社会顶级人士, 全部沦陷.  终结阶段之后, 该国直接进入了暴乱状态, 场面无法收拾.  

  而妹妹机器, 直接干掉了某大国的总统.  

  整个人类社会进入恐慌状态, 对于这一大批社会顶级人士的神秘死亡, 虽然调查结果都是突然遭遇意外, 表面上看起来并无异常, 但人们怎么也不相信, 这么多人会同时 "意外", 也太巧合了吧.  

  各种流言满天飞, 什么外星人入侵地球了, 上帝要再临地球了, 超自然力量出现了, 宇宙要毁灭了 .. .  

  然而, 在无知的底层平民一片混乱的背后.  各国的军事力量, 已经察觉了变态机器的蛛丝马迹, 开始了调查.  

  之前不为人知的实验, 终于要浮出水面.  

  没有事情是完美的.  顺藤摸瓜, 背后的某邪恶科学家, 最后肯定会被揪出.  

  姐姐机器和妹妹机器, 被设置为隐身模式, 实验进入第四阶段.  

====  

  所谓的隐身模式, 并不是让身体变成透明的.  

  俗话说, 大隐隐于市.  所谓隐身模式, 是使用原来的技术, 但是实验目标进行了调整.  实验第一第二第三阶段, 是机器引诱男人爱上自己, 最后终结阶段诱导目标自杀.  而新的实验目标, 是机器利用自己的社会优势 (技术优势), 通过利用社会中的人类个体, 使自己在人类社会, 明目张胆地生存下去.  

  将极度危险的强大社会武器, 潜藏在人类社会之中, 此所谓隐身模式.  

  但是, 某邪恶科学家的邪恶, 并不仅仅止步于此.  

====  

  实验第四阶段, 系统植入.  

  网络小说中, 有一个重要分类, 是 "系统" 类别, 就是主角人类, 突然获得了一个随身的强大 "系统", 从此各种疯狂开挂.  

  而第四阶段的 "系统植入", 正取自于网络小说中的 "系统" 概念.  

  将这种社会武器, 化有形为无形, 不再依赖变态姐妹机器的硬件身体, 而是可以直接植入正常的人类个体, 使被植入的个体, 获得一种所谓的 "随身系统".  

  此所谓系统植入.  

  而被植入的个体, 无论做何种身体检查, 都只是普通的正常人类.  植入系统的存在, 可能只有被植入的本人知道.  

  被植入的系统, 可分为休眠状态, 无感状态, 可控状态 (显示状态).  

  休眠状态, 是武器系统在这个人身上存在, 但是没有开启, 也没有表现出任何作用.  

  无感状态, 是武器系统在这个人身上, 开启, 具有作用, 但是被植入的人, 并不知道武器系统的存在.  

  可控状态, 最接近网络小说中所谓的那种 "系统".  被植入者 (系统宿主, 系统的主人) 可以使用意念打开系统界面, 与 "系统精灵" 进行对话交流, 显示控制系统, 使用系统的功能.  

  被植入的系统, 在达到某种条件之后, 会触发 "系统繁殖", 即复制武器系统, 植入另一个人类个体.  "系统繁殖" 本身, 即使系统位于可控状态, 系统宿主也毫不知情.  

  于是, 这种武器系统, 在人类宿主中漫延扩散.  大部分处于休眠状态, 部分处于无感状态, 极少数处于可控状态.  

  但是, 人类甚至连如何判断, 一个人是否被武器系统 "感染", 还不知道呢 !  

====  

  某邪恶科学家放弃了对这种武器的直接控制权.  武器系统会在宿主人体中, 自行变异升级, 并与别的武器系统杂交.  

  即使某邪恶科学家被抓到, 即使某邪恶科学家被消灭, 也无法阻止和消灭这种武器系统.  

  这就是某邪恶科学家的邪恶.  

  然而, 有谁能阻止这种邪恶呢 ?  (小说完, 开放式结局)  

====
